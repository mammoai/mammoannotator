Instructions for annotating MRI images

You will be presented with up to four images tha come from the same patient and examination study.

Do not paint any pixels if the image is wrong or there is any issue, please write the description of the issue ("Wrong cropping", "Saturated", etc.) in the Annotation Comments then click "Add" or use the checkboxes if the problem is described by them. Finally click "Submit"

The order of the images is as follows:

Right Breast
Sagittal 		Left Breast
Sagittal
Right Breast
Axial 		Left Breast
Axial

Most of the tools and labels are accessible thorugh the keyboard.
1-9 Select label.
ESC for deselect the current region. Use it before changing the label, otherwise it will change the current region's label.
B Brush tool.
E Eraser tool.
CTRL+Z for undo.
CTRL+mouse point & scroll for zoom.

To create an annotation:

  1. Select a label (keyboard 1-9 or click on the label)
  2. Paint the pixels that should be labeled.
  3. Press ESC
